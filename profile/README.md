## ⭐ 기획 배경
- 책에서 발견한 인상적인 구절, 수업 중의 필기, 메모장의 기록들은 시간이 지날수록 희미해지고 결국 손에서 멀어지게 됩니다.
- 이러한 소중한 기록들을 영구적으로 보존하고, 서로 연결하여 새로운 인사이트를 발견할 수 있는 플랫폼을 제공합니다.
